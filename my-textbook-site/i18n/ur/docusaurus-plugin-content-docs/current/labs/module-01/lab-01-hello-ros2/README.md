# لیب 1: ہیلو ROS 2

**مدت**: 60 منٹ
**درس**: [درس 3: rclpy کے ساتھ پائی تھن ایجنٹس](/docs/module-01-ros2/lesson-03-rclpy-agents)
**ضروریات**: ROS 2 ہمبل انسٹال، پائی تھن 3.10+

## اہداف

اپنا پہلا ROS 2 نوڈ بنائیں اور چلائیں. اس لیب کے اختتام تک، آپ ROS 2 پائی تھن نوڈ کی بنیادی ساخت اور اس کی تصدیق کرنا سیکھیں گے کہ یہ صحیح طریقے سے چل رہا ہے.

## سیکھنے کے نتائج

اس لیب کو مکمل کرنے کے بعد، آپ کر پائیں گے:
- ROS 2 ورک سپیس سیٹ کرنا
- ROS 2 پائی تھن پیکج بنانا
- ایک منیمال نوڈ لکھنا جو پیغامات لاگ کرتا ہے
- ROS 2 CLI ٹولز کا استعمال کرکے نوڈ کی تصدیق کرنا کہ یہ چل رہا ہے

## سیٹ اپ

### 1. ورک سپیس بنائیں

```bash
mkdir -p ~/ros2_ws/src
cd ~/ros2_ws/src
```

### 2. پیکج بنائیں

```bash
ros2 pkg create --build-type ament_python hello_ros2 --dependencies rclpy
```

### 3. پیکج میں جائیں

```bash
cd hello_ros2/hello_ros2
```

## کام

### کام 1: نوڈ لکھیں (20 منٹ)

`hello_ros2/hello_ros2/` ڈائریکٹری میں `hello_node.py` فائل بنائیں.

**ضروریات:**
- `Node` سے وراثت میں `HelloNode` نام کا کلاس بنائیں
- نوڈ کا نام `'hello_ros2_node'` ہونا چاہیے
- نوڈ شروع ہونے پر "Hello, ROS 2!" لاگ کرنا چاہیے
- ہر 2 سیکنڈ میں "Still running..." لاگ کرنے کے لیے ٹائمر استعمال کریں

**اسٹارٹر کوڈ `starter/hello_node.py` میں فراہم کیا گیا ہے**

### کام 2: پیکج کی ترتیب (10 منٹ)

اینٹری پوائنٹ شامل کرنے کے لیے `setup.py` میں ترمیم کریں:

```python
entry_points={
    'console_scripts': [
        'hello_node = hello_ros2.hello_node:main',
    ],
},
```

### کام 3: تعمیر اور چلائیں (15 منٹ)

```bash
# ورک سپیس کے روٹ میں جائیں
cd ~/ros2_ws

# پیکج تعمیر کریں
colcon build --packages-select hello_ros2

# ورک سپیس کا ذریعہ بنائیں
source install/setup.bash

# نوڈ چلائیں
ros2 run hello_ros2 hello_node
```

### کام 4: تصدیق (15 منٹ)

ایک نئے ٹرمنل میں:

```bash
# ROS 2 کا ذریعہ بنائیں
source /opt/ros/humble/setup.bash
source ~/ros2_ws/install/setup.bash

# چلنے والے نوڈز کو فہرست کریں
ros2 node list

# نوڈ کی معلومات چیک کریں
ros2 node info /hello_ros2_node
```

## توثیق

اپنے نفاذ کی تصدیق کے لیے توثیق اسکرپٹ چلائیں:

```bash
python3 validate.py
```

متوقع آؤٹ پٹ:
```
✅ نوڈ 'hello_ros2_node' ملا
✅ نوڈ لاگز شائع کر رہا ہے
✅ تمام چیکس پاس!
```

## متوقع نتائج

1. `ros2 node list` `/hello_ros2_node` دکھاتا ہے
2. ٹرمنل "Hello, ROS 2!" کے بعد ہر 2 سیکنڈ میں "Still running..." ڈسپلے کرتا ہے
3. `ros2 node info /hello_ros2_node` نوڈ کی تفصیل دکھاتا ہے

## مسئلہ حل

| مسئلہ | حل |
|-------|----------|
| نوڈ نہیں ملا | یقینی بنائیں کہ آپ نے ورک سپیس کا ذریعہ بنایا |
| امپورٹ کی غلطی | rclpy انسٹال ہے چیک کریں: `pip3 install rclpy` |
| اجازت سے انکار | پائی تھن فائل کو ایگزیکوٹیبل بنائیں: `chmod +x hello_node.py` |
