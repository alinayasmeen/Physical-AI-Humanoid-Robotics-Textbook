# مینی پروجیکٹ 2: اعلی وژن-زبان-عمل (VLA) سسٹم

## جائزہ
اس اعلی مینی پروجیکٹ میں، آپ اپنے بنیادی VLA سسٹم کو وسعت دیں گے تاکہ وہ اشیاء کی آگاہی اور اعلی سیفٹی مانیٹرنگ کے ساتھ زیادہ پیچیدہ ملٹی موڈل کمانڈز کو ہینڈل کر سکے. یہ پروجیکٹ مکمل VLA سسٹم کے انضمام اور حقیقی دنیا کی ڈیپلائمنٹ کے امور کو سمجھنے میں گہرائی بخشدے گا.

## سیکھنے کے اہداف
- وژن انضمام کے ساتھ اشیاء کی آگاہی VLA پروسیسنگ نافذ کریں
- LLMs کا استعمال کرتے ہوئے اعلی کوگنیٹو پلاننگ ڈیزائن کریں
- جامع سیفٹی اور غلطی کے انتظام کو نافذ کریں
- حقیقی وقت کے عمل کے لیے VLA سسٹم کی کارکردگی کو بہتر بنائیں
- سسٹم کی کارکردگی اور حدود کا جائزہ لیں

## ضروریات
- مینی پروجیکٹ 1 (سادہ VLA سسٹم) مکمل
- ROS 2 ایکشن انٹرفیسز کی سمجھ
- کمپیوٹر وژن اور اوبجیکٹ ڈیٹیکشن کے بارے میں بنیادی علم
- LLM انضمام کی واقفیت (موڈیول 4 میں زیر بحث)

## کام کی تقسیم

### 1. وژن پروسیسنگ جزو
اعلی وژن پروسیسنگ نافذ کریں جو یہ کر سکے:

- ماحول میں اشیاء کو ڈیٹیکٹ اور کلاسیفائی کریں
- اشیاء کی پوزز اور جگہی تعلقات کا تخمینہ لگائیں
- منصوبہ بندی کے سسٹم کو اشیاء کی آگاہی کا سیاق و سباق فراہم کریں

### 2. کوگنیٹو پلاننگ جزو
LLM-بیسڈ منصوبہ بندی نافذ کریں جو یہ کر سکے:

- متعدد اقدامات کی ایکشن سیکوئنس جنریٹ کریں
- غیر واضح یا پیچیدہ کمانڈز کو ہینڈل کریں
- ماحولیاتی پابندیوں اور اشیاء کی خصوصیات پر غور کریں

### 3. اعلی سیفٹی مانیٹرنگ
جامع سیفٹی سسٹم نافذ کریں جو:

- سیفٹی کی پاسداری کے لیے تمام ایکشنز کو مانیٹر کریں
- غیر متوقع صورتحال کو باوقار طریقے سے ہینڈل کریں
- ناکامیوں کے لیے فال بیک میکنزم فراہم کریں

### 4. کارکردگی کی بہتری
مکمل سسٹم کو مندرجہ ذیل کے لیے بہتر بنائیں:

- حقیقی وقت کی پروسیسنگ کی ضروریات
- کارآمد وسائل کا استعمال
- مختلف حالات میں مستحکم آپریشن

## شروع کریں
1. `starter/` ڈائریکٹری میں جائیں
2. اپنے پچھلے مینی پروجیکٹ کے نفاذ پر کام کریں
3. فراہم کردہ فائلز میں اعلی فعلیت نافذ کریں
4. `validate.py` اسکرپٹ کے ساتھ اپنے نفاذ کو ٹیسٹ کریں
5. مکمل ہونے پر `solution/` ڈائریکٹری کے ساتھ موازنہ کریں

## توثیق
اپنے نفاذ کی جانچ کے لیے توثیق اسکرپٹ چلائیں:

```bash
python validate.py
```
