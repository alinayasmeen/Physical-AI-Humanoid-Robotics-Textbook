# مینی پروجیکٹ 1: سادہ وژن-لینگویج-ایکشن (VLA) سسٹم

## جائزہ
اس مینی پروجیکٹ میں، آپ ایک بنیادی وژن-لینگویج-ایکشن (VLA) سسٹم تیار کریں گے جو سادہ بولی گئی کمانڈز کو سمجھ سکتا ہے اور روبوٹ سیمولیٹر کا استعمال کرتے ہوئے انہیں انجام دے سکتا ہے. یہ پروجیکٹ آپ کو VLA سسٹم کے بنیادی اجزاء اور ان کے انضمام کو سمجھنے میں مدد دے گا.

## سیکھنے کے اہداف
- بولی کی پہچان، وژن، اور ایکشن اجزاء کے ساتھ ایک بنیادی VLA پائپ لائن نافذ کریں
- ملٹی موڈل ای آئی اجزاء کے ساتھ ROS 2 کو انضمام کریں
- بنیادی غلطی کے کیسز اور حفاظتی اقدامات کو ہینڈل کریں
- "move forward" یا "turn left" جیسے سادہ کمانڈز کے ساتھ سسٹم کو ٹیسٹ کریں

## ضروریات
- موڈیول 1-3 مکمل (ROS 2، سیمولیشن، پرچیپشن)
- بنیادی پائی تھن پروگرامنگ کی مہارتیں
- موڈیول 4 سے ملٹی موڈل ای آئی تصورات کی سمجھ

## تخمینی وقت
4-6 گھنٹے


## کام کی تقسیم

### 1. بولی کی پہچان کا جزو
ایک بنیادی بولی کی پہچان کا ماڈیول نافذ کریں جو یہ کر سکے:
- آڈیو ان پٹ کو قبضہ کریں
- بولی کو متن میں تبدیل کریں
- سادہ کمانڈز کو پارس کریں

### 2. بنیادی ایکشن انجام دہی
ROS 2 ایکشن انجام دہی نافذ کریں جو یہ کر سکے:
- سادہ حرکت کمانڈز انجام دیں
- بنیادی نیویگیشن کے کاموں کو ہینڈل کریں
- انجام دہی کی حالت کو مانیٹر کریں

### 3. پائپ لائن انضمام
تمام اجزاء کو ایک کام کرتی ہوئی پائپ لائن میں جوڑیں جو:
- بولی گئی کمانڈز کو عمل کرے
- مناسب ایکشنز انجام دے
- صارف کو فیڈ بیک فراہم کرے

## شروع کریں
1. `starter/` ڈائریکٹری میں جائیں
2. فراہم کردہ فائلوں میں غائب فعالیت نافذ کریں
3. `validate.py` اسکرپٹ کے ساتھ اپنے نفاذ کو ٹیسٹ کریں
4. مکمل ہونے پر `solution/` ڈائریکٹری کے ساتھ موازنہ کریں

## توثیق
اپنے نفاذ کی جانچ کے لیے توثیق اسکرپٹ چلائیں:
```bash
python validate.py
```
