# عملی لیب تربیت

فزیکل ای آئی اور ہیومنوائڈ روبوٹکس کورس کے عملی جزو میں خوش آمدید. یہ لیبز ہر چیپٹر میں شامل تصورات کے ساتھ عملی تجربہ فراہم کرتی ہیں۔

## لیب کی ساخت

ہر لیب میں شامل ہے:

- **اہداف**: آپ کیا کریں گے
- **ضروریات**: ضروری علم اور پچھلی لیبز
- **تقریبی وقت**: 60-180 منٹ
- **اسٹارٹر کوڈ**: ٹوڈوز کو مکمل کرنے کے ساتھ ٹیمپلیٹس
- **حل**: حوالہ نفاذ
- **توثیق اسکرپٹ**: کامیابی کی تصدیق کے لیے خودکار ٹیسٹنگ

## شروع کریں

### ماحول کا سیٹ اپ

لیبز شروع کرنے سے پہلے، یقینی بنائیں کہ آپ کے پاس ہے:

1. **ROS 2 ہمبل** اوبنٹو 22.04 پر انسٹال
2. **پائی تھن 3.10+** پائپ کے ساتھ
3. **گیزبو کلاسک 11** (ROS 2 ڈیسک ٹاپ کے ساتھ شامل)
4. مناسب طریقے سے تشکیل دیا گیا **ورک سپیس**

### لیبز چلانا

ہر لیب ڈائریکٹری میں ہے:

```text
لیبز/لیب-XX-XX-نام/
├── README.md          # لیب ہدایات
├── اسٹارٹر/           # شروع کا کوڈ (ٹوڈوز کو مکمل کریں)
├── حل/          # حوالہ نفاذ
└── validate.py        # اپنے کام کی جانچ کے لیے چلائیں
```

### توثیق

لیب مکمل کرنے کے بعد، توثیق اسکرپٹ چلائیں:

```bash
cd لیبز/لیب-XX-XX-نام
python validate.py
```

سبز چیک مارکس کامیابی کی نشاندہی کرتے ہیں. لال X نشان دیکھتے ہیں کہ کیا درست کرنے کی ضرورت ہے.

## کامیابی کے لیے نکات

1. **پہلے README پڑھیں** - کوڈنگ سے پہلے اہداف کو سمجھیں
2. **مرحلے کے مطابق م follow کریں** - لیبز پچھلے تصورات پر تعمیر کرتی ہیں
3. **اکثر توثیق چلائیں** - جلد جلد غلطیاں پکڑیں
4. **توسیع کی کوشش کریں** - اختیاری چیلنجز کے ساتھ اپنی سمجھ بڑھائیں
5. **سوالات پوچھیں** - اگر پھنس گئے، تو چیپٹر کا مواد دیکھیں یا اپنے استاد سے پوچھیں

تیار ہیں شروع کرنے کے لیے؟ فصل 2 مکمل کرنے کے بعد اپنی پہلی لیب شروع کریں!
