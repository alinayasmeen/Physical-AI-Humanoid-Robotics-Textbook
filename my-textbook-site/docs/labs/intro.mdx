# Hands-On Lab Exercises

Welcome to the practical component of the Physical AI & Humanoid Robotics course. These labs provide hands-on experience with the concepts covered in each chapter.

## Lab Structure

Each lab includes:

- **Objective**: What you will accomplish
- **Prerequisites**: Required knowledge and prior labs
- **Estimated Time**: 60-180 minutes
- **Starter Code**: Templates with TODOs to complete
- **Solution**: Reference implementation
- **Validation Script**: Automated testing to verify success

## Getting Started

### Environment Setup

Before starting the labs, ensure you have:

1. **ROS 2 Humble** installed on Ubuntu 22.04
2. **Python 3.10+** with pip
3. **Gazebo Classic 11** (included with ROS 2 Desktop)
4. A properly configured **workspace**

### Running Labs

Each lab directory contains:

```text
labs/lab-XX-XX-name/
├── README.md          # Lab instructions
├── starter/           # Starting code (complete the TODOs)
├── solution/          # Reference implementation
└── validate.py        # Run this to check your work
```

### Validation

After completing a lab, run the validation script:

```bash
cd labs/lab-XX-XX-name
python validate.py
```

Green checkmarks indicate success. Red X marks show what needs fixing.

## Tips for Success

1. **Read the README first** - Understand the objective before coding
2. **Follow the steps in order** - Labs build on previous concepts
3. **Run validation often** - Catch errors early
4. **Try the extensions** - Deepen your understanding with optional challenges
5. **Ask questions** - If stuck, review the chapter content or ask your instructor

Ready to begin? Start with your first lab after completing Chapter 2!
